About BS.JS

BeatSaver.JS (BS.JS) is a JavaScript wrapper library developed by Lufthor

This wrapper has a total of 11+ functions to make using the BeatSaver API easier

The BS.JS Wrapper is currently on version 1.2.0

There is an already existing wrapper, however this one is more light-weight

Creator: Lufthor

Contributors: NONE
