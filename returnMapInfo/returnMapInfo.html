Returning Map Information

Q: What is returnMapInfo?

A: returnMapInfo is a function that can be used to return the information of a map.

▼ How to use ▼

            
                const bs = require('bs.js');

                            #Reminder -> These functions can only be used in async functions
                async function onStart() {
                                const varName = await bs.returnMapInfo(69420)
                                #To see all the cool API json stuff you can use, write this
                        console.log(varName)
}

Parameters

1. MapID -- The id of the map to search
